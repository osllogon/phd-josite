## Parallel Node-Level Explainability for Graph Neural Networks (GNNs)

In this talk, I explained the research I have been pursuing as part of my PhD about parallelizing the explainability for GNNs. This research has been published as a [paper](https://arxiv.org/abs/2601.04807).

## PhD - Scalable Post-Hoc Explainability for Graph Neural Networks

The main topic of this thesis is how to make scalable what is known in the literature as Post-Hoc Explainability for Graph Neural Networks.
